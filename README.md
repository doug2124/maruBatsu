# 三目並べ（〇×ゲーム）Web版

HTML / CSS / JavaScript を使って実装したブラウザ版の三目並べゲームです。  
DOM 操作、イベント処理、ゲームロジックの基礎を練習するために作成しました。

## 🛠 使用技術
- HTML5
- CSS3
- JavaScript（バニラ）

## ▶ 実行方法
`index.html` をブラウザで開くだけでプレイできます。

## 📌 主な機能
- 3×3 のマスをクリックして X / O を配置
- 勝敗判定（縦・横・斜め）
- 引き分け判定
- リセットボタンで再スタート

## 🤖 コンピューターのロジック
コンピューターの手は「空いているマスからランダムに選ぶ」というシンプルなアルゴリズムを使用しています。  
高度な戦略（勝ち筋の探索、ブロックなど）は未実装です。

## 🧠 学んだこと
- DOM の操作方法
- クリックイベントの処理
- 配列を使った勝敗判定ロジック
- シンプルな UI の構築

## 🚀 今後の改善点
- ランダムではなく、簡易 AI の実装
- minimax アルゴリズムによる最適手の計算
- 難易度設定（Easy / Hard）
